# Typescript-template

## Start project
npm init

entry point: index.js
test command: jest

## Install typscript
npm i typescript -D

npx tsc --init

## Add ESLint
npm i eslint -D
npx eslint init

> rules: {
> "quotes": ["error", "double"]
> }

## Install prettier
npm i --save-dev eslint-config-prettier

### Test file
npx tsc src/filename.ts
